## 项目经历

### CellSAM 适配：面向 hiPSC-CM 的全细胞实例分割项目

硕士毕业设计 / 导师实验室项目

- 围绕 hiPSC-derived cardiomyocytes 密集显微图像中的弱边界、细胞粘连和标注不完整问题，定义 whole-cell instance segmentation 任务，并将问题拆解为分割能力与自动提示生成能力两类瓶颈。
- 设计 split-aware 评估思路，将 GT-box Oracle 评估与 fully automatic E2E 推理分离，避免把 mask decoder 能力和 prompt generation 质量混在一起判断。
- 基于 BF、Actn2、DAPI 三通道生物学先验，推动 CellSAM 从通用细胞分割模型适配到心肌细胞特定场景，并通过固定评估协议支持方案对比和瓶颈定位。
- 对比 CellSAM automatic anchor routing、Cellpose 等 baseline，验证未适配流程在该场景下表现有限，并明确自动部署的主要瓶颈来自 prompt generation / candidate generation。
- 在 locked test73 上，三通道 decoder 在 Oracle 设置下达到 PQ 0.691 / BM-Dice 0.819 / AJI 0.698 / F1 0.981；自动 E2E DAPI-CM pipeline 达到 F1 0.617 / PQ 0.398。
